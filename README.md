# Magic Quest
#MD_FILES


### Clone project
```bash
git clone git@github.com:eatae/magic-quest.git && cd magic-quest
```

### Build
```bash
sh command build
```

### Start questions
```bash
sh command questions
```

### Show saved result
```bash
sh command result
```

### Down containers
```bash
sh command docker:down
```
